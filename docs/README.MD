## Getting Started

### Prerequisites

- Python 3.11.3
- Docker 24.0.2

### Installation

1. Get an API Key from [API Ninjas](https://api-ninjas.com/)

1. Clone the repo:
   git clone https://gitlab.com/goldnine/sculptr-app.git

1. Navigate to the directory:
   cd sculptr-app

1. Set up Docker:
   docker-compose up --build
